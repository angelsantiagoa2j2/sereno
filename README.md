# 🏡 SERENO INMOBILIARIA

**Sereno** es una plataforma web inmobiliaria desarrollada para centralizar la gestión de propiedades, clientes e inmobiliarias en un solo sistema digital.

Construida con tecnologías como **Java EE (Servlets + JSP)** y **MySQL**, permite gestionar todo el flujo de una inmobiliaria: desde la visualización de propiedades hasta la coordinación de citas.

---

## 🚀 ¿Qué es Sereno?

Sereno es una solución web que conecta tres tipos de usuarios:

- 👤 **Clientes** → buscan propiedades, envían solicitudes y agendan citas  
- 🏢 **Inmobiliarias** → publican propiedades y gestionan solicitudes  
- ⚙️ **Administradores** → controlan usuarios, accesos y el sistema  

Todo esto dentro de una plataforma centralizada, segura y organizada.

---

## 🧠 Enfoque del proyecto

El desarrollo se realizó utilizando la metodología **Scrum**, con:

- 📅 **3 Sprints**
- ⏱️ **1 semana por sprint**
- ✅ **100% del backlog completado**

Cada sprint incluyó:
- Planning  
- Desarrollo  
- Testing  
- Review  
- Retrospective

## ⚙️ Funcionalidades principales

### 🔐 Autenticación
- Registro de usuarios
- Inicio de sesión seguro
- Control de acceso por roles

---

### 🏡 Gestión de propiedades
- Crear, editar y eliminar propiedades
- Imágenes, precio, ubicación y descripción
- Listado público de inmuebles

---

### 🔍 Búsqueda avanzada
- Filtros por:
  - Tipo
  - Ubicación
  - Precio
- Resultados dinámicos

---

### 👥 Gestión de usuarios (Admin)
- CRUD de usuarios
- Cambio de roles
- Control de accesos

---

### 📩 Solicitudes
- Clientes pueden enviar solicitudes
- Inmobiliarias gestionan estados

---

### 📅 Sistema de citas
- Agendar visitas a propiedades
- Estados: pendiente, confirmada, cancelada

---

### 📊 Dashboards personalizados
Cada usuario tiene su propio panel:

- 👤 Cliente → solicitudes y citas  
- 🏢 Inmobiliaria → propiedades y clientes  
- ⚙️ Admin → control total del sistema
